# Admin Tools

Administrative tools for managing the MidOS knowledge base, users, and system configuration. These tools require admin-level access.

## Available Tools

Admin tools provide capabilities for:

- **Knowledge Management** — Ingest, update, and curate knowledge chunks
- **User Management** — Manage API keys, tiers, and access control
- **System Configuration** — Configure search parameters, embeddings, and pipelines
- **Analytics** — View usage statistics, query patterns, and system health
